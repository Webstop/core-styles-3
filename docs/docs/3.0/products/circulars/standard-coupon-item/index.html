The optional col class allows circular items align nicely in a flex box grid. Circular items which rest inside a row class and contain a col class will nicely divide up the available space in a row to provide a nice group presentation.

If a circular item is going to stand alone on a page or section of a page, the col class is unnecessary.

Grid of Coupon Circular Items

This layout is achieved by giving the circular items a col class within the context of a row.
