Ajax Load

An easy way to load HTML content into a section of the page.

li class= a href=/articles/ Articles ul li class=active a href=/docs/ Articles ul li class= a href=/guides/ Guides ul li class= a href=/resources/ resources ul

Examples

Here are a couple of common uses of the AJAX Load feature.

Basic Example

To use the Ajax Load feature place a data-ajax attribute on an HTML element (e.g. link or button). The data-load attribute specifies the URL to pull the content from, it is expected to be in HTML format. You specify where to place the AJAX content with the data-target attribute, it accepts standard jQuery selectors.

This box could be replaced by AJAX content, if only someone would click the button above. 
1<button data-ajax-load data-target="#ajax-target-1" data-load="{{ sinatra_host }}/alert_success" class="btn btn-primary mb-2">
2  Load Content
3</button>
4
5<div id="ajax-target-1">
6  <div class="alert alert-info">
7    This box could be replaced by AJAX content, if only someone would click the button above.
8  </div>
9</div>

Replacement Example

In the following example we replace the button that triggers the request with the content returned by the AJAX call.

1<div id="ajax-target-replace">
2  <button data-ajax-load data-target="#ajax-target-replace" data-load="{{ sinatra_host }}/alert_success" class="btn btn-primary">
3    Replace Me with AJAX Content!
4  </button>
5</div>

Attributes

The following attributes are required.

Options Description
data-ajax-load The presence of this attribute indicates clicking on this element should trigger an AJAX load.
data-load Attribute specifies the URL of the content you want AJAX'd. The AJAX request should return HTML content to display inside the target element.
data-target The DOM node to load the content into. Uses standard jQuery selectors, usually targets an id attribute (e.g. #some-target).
data-power-bar When present, completing the ajax call triggers a reload of the Shopping List Power Bar.
