Ajax Modal

A modal available site-wide for AJAX content.

The site-model is included on every page when you used the code found on the site layout page.

Example

When you set the data-target of a modal trigger to #site-modal, then you are indicating you want to use the site wide modal, which enables remote content loading via AJAX. The data-load attribute specifies the URL to pull the body content from, it is expected to be in HTML format. You should also include the data-title to set the text for the title of the modal.

Options

The Site Modal inherits all the abilities of a standard modal, and is extended with the following options.

Options Description
data-title Attribute used to set the title of the modal.
data-load Attribute specifies the URL of the content you want AJAX'd into the modal body. The AJAX request should return HTML content to display inside the modal.
data-footer By default the modal will display a footer with a close button. To hide the footer and it's content, include the attribute data-footer set to hide (data-footer="hide") on the trigger element.
